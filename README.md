A catalog of my STA 301H: Introduction to Data Science Program Assignments at the University of Texas Austin 

## Overview
This repository contains my R scripts, analyses, and coursework from STA 301H at the University of Texas at Austin.

The materials here demonstrate foundational and applied skills in:
- Statistical reasoning and data interpretation
- Programming in R for data analysis and visualization
- Probability modeling and inference
- Reproducible data workflows and analytical reporting

This repository serves as a learning archive and professional portfolio showcasing my growth in statistical computing and quantitative analysis.

### Course Goals
The primary objectives of STA 301H include:
- Developing statistical intuition for real-world data
- Learning to write clear, correct, and efficient R code
- Applying probability distributions, hypothesis testing, and regression
- Creating data visualizations that communicate insights effectively
- Practicing reproducible and ethical data analysis
- These goals support broader preparation in:
- Engineering data analysis
- Business analytics and decision-making
- Machine learning and quantitative modeling
